TimeStamp API

This is a simple API app using node.js to achieve the function that display date query into two format:

  • Unix
  • Natural human readable format

Here is the example query:

natural language:

https://timestampproject-jess901108.c9users.io/March 16, 2016

or UNIX format

https://timestampproject-jess901108.c9users.io/1458086400

Will both give output like this:

{'natural': "March 16, 2016", 'unix': "1458086400"}
